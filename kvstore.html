// FEATURES

What's actually inside.

[O(1)]

LRU Eviction

Hash map gives instant lookup; doubly linked list moves nodes to front and evicts from the back, both in constant time.

[TTL]

Min-Heap Expiry

Keys with a TTL sit in a min-heap ordered by expiry time, so the soonest-to-expire key is always reachable without scanning the store.

[x8]

Concurrent Workers

An 8-thread pool handles parallel client connections, with mutex-guarded critical sections kept deliberately small.

[TCP]

Custom Protocol

A lightweight protocol built directly on POSIX sockets — no framework, no abstraction layer between the wire and the logic.

[ASAN]

Memory Safety

Validated with AddressSanitizer across 10,000+ concurrent requests. Zero leaks, zero use-after-free.

[□]

Containerized

Ships with a Dockerfile — one command builds and runs the full server in an isolated environment.

Run it yourself.

# clone and build with docker
git clone https://github.com/Omkargarde123/KVSTORE.git
cd KVSTORE
docker build -t kvstore .
docker run -p 6379:6379 kvstore

# talk to it
echo "SET foo bar" | nc localhost 6379
echo "GET foo" | nc localhost 6379
